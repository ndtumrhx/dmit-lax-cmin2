## LAX.EB.WEE 网络线路：三网 CMIN2 是什么水平？

这才是大家最关心的部分。

DMIT LAX.EB 系列的网络特性，用一句话概括就是：**去程 BGP 直连，回程三网走 CMIN2 高速线路**。

### 回程线路拆解

- **电信回程**：走 AS9929（CTGNet）优质线路，低延迟、低抖动
- **联通回程**：走 AS9929，同样是联通的精品骨干网
- **移动回程**：走 CMIN2（中国移动国际精品网络），这是移动近年来主推的高质量国际线路

CMIN2 是中国移动专门针对国际流量打造的精品网络，跟普通的移动国际线路（CMI）不是一回事。CMIN2 的特点是路由更短、延迟更稳、高峰时段掉速更少，通俗说就是晚上高峰期也不容易卡。

### 实测延迟参考

洛杉矶到中国大陆的延迟，受地理距离影响，基准在 150~180ms 左右，这是物理极限无法突破的范围。测试 IP：`154.17.226.2`，有兴趣可以自己 ping 一下感受延迟。

> **一个真实的感受**：跟普通走 CMI 或者 AS4837 的线路相比，LAX.EB.WEE 的晚高峰表现稳定多了。不是说 0 抖动，但掉速幅度明显可控，跑满不了 1Gbps，但几十 Mbps 的稳定带宽基本能保持。
